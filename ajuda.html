Ajuda do Sistema de Fichas de Indicadores de Desempenho

Este sistema foi desenvolvido para apoiar o cadastro estruturado dos indicadores de desempenho, o relacionamento com objetivos estratégicos, o registro de metas anuais e a geração de relatórios para acompanhamento da execução da estratégia.

Visão geral das abas

  • GESTOR RESPONSÁVEL Cadastro e edição dos detalhes técnicos dos indicadores sob responsabilidade de cada gestor.
  • ADMIN Administração geral do sistema (gestores, perspectivas, objetivos, indicadores e controle de edição).
  • METAS Cadastro de metas anuais por indicador, definição de faixas de gestão à vista e valores de referência.
  • ORIENTAÇÕES Conceitos e instruções sobre cada campo da ficha de indicadores.
  • RESULTADOS Relatórios consolidados dos indicadores cadastrados, com opções de filtro e exportação.

1. Aba “GESTOR RESPONSÁVEL”

Nesta aba, cada gestor seleciona seu nome na lista e visualiza os indicadores sob sua responsabilidade.

  1. Selecione o seu nome no campo “Nome do Gestor Responsável”.
  2. Na tabela de indicadores, clique em “Preencher / Editar detalhes” para o indicador desejado.
  3. Preencha os campos:
    • Descrição operacional
    • Fórmula
    • Unidade de medida
    • Fonte de dados
    • Periodicidade
    • Polaridade
  4. Use “Salvar rascunho” para salvamento parcial (edição livre pelo gestor).
  5. Use “Salvar definitivo” para travar todos os indicadores completos em rascunho do gestor (somente o ADMIN poderá liberar para nova edição).

Após o salvamento definitivo, os dados passam a ser usados como base oficial para relatórios e monitoramento.

2. Aba “ADMIN”

Somente pessoas com a senha do ADMIN podem cadastrar e manter a estrutura do sistema.

  1. Informe a senha no campo “Senha do ADMIN” e clique em “Entrar / Revalidar senha”.
  2. Cadastre:
    • Gestores responsáveis
    • Perspectivas (ex.: Financeira, Clientes, Processos, Pessoas, etc.)
    • Objetivos estratégicos vinculados à perspectiva e ao gestor
    • Indicadores vinculados a cada objetivo estratégico
  3. Na tabela de indicadores, o ADMIN pode:
    • Editar o nome do indicador
    • Excluir indicadores (desde que não estejam em uso por metas ou outros controles internos do usuário)
    • Liberar edição de indicadores que estejam como “Definitivos”
  4. Na tabela de Metas dos Indicadores (ADMIN), o ADMIN pode:
    • Visualizar todas as metas cadastradas (por perspectiva, objetivo, indicador, gestor e ano)
    • Excluir metas cadastradas, caso seja necessário refazer o cadastro
    • Liberar edição de metas definitivas, retornando-as ao status de rascunho para novo ajuste pelo gestor
  5. Ao concluir a configuração inicial, usar o botão “Validar cadastro” para encerrar a sessão de administração.

3. Aba “METAS” – Cadastro de Metas dos Indicadores

Nesta aba, os gestores registram as metas do indicador para um ano de monitoramento, definem o tipo de cálculo, as faixas de gestão à vista e os valores de referência dos últimos anos.

3.1. Passo a passo para cadastrar ou editar metas

  1. Selecione o seu nome em “Nome do Gestor Responsável”.
  2. Localize o indicador desejado na tabela e clique em “Cadastrar / Editar metas”.
  3. Preencha:
    • Ano de monitoramento (ano das metas) – Ex.: 2026.
    • Metas mensais de janeiro a dezembro, respeitando a unidade de medida do indicador (número, %, R$, etc.).
    • Tipo de cálculo:
      • Indicador Mensal (mês isolado) – analisa apenas o valor daquele mês.
      • Indicador Acumulado (YTD – soma) – soma os resultados de janeiro até o mês corrente.
      • Indicador Acumulado (YTD – média) – calcula a média dos resultados de janeiro até o mês corrente.
    • Faixas de gestão à vista (% de atingimento da meta):
      • Azul: de X até Y
      • Verde: de X até Y
      • Amarelo: de X até Y
      • Vermelho: abaixo de X
      As faixas devem ser crescente e sem sobreposição, para garantir leitura visual consistente.
    • Valores de referência (últimos 3 anos) – opção para informar os valores realizados do indicador nos três anos anteriores ao ano de monitoramento. Estes valores são usados para contextualizar as metas.
  4. Use “Salvar rascunho de metas” enquanto ainda estiver ajustando os valores.
  5. Use “SALVAR DEFINITIVO DAS METAS” quando estiver seguro de que as metas estão consolidadas:
    • As metas passam ao status “Definitiva”.
    • A edição fica bloqueada para o gestor.
    • Apenas o ADMIN poderá liberar novamente para edição ou excluir a meta.

3.2. Relatório de metas e exportação

No final da aba METAS há um relatório consolidado das metas cadastradas.

  • Filtros por Perspectiva, Objetivo Estratégico e Indicador.
  • Botão “Imprimir / Exportar PDF” – utilize a opção “Salvar como PDF” do navegador.
  • Botão “Exportar metas para Excel (CSV)” – gera arquivo CSV em UTF-8 com BOM, garantindo a correta exibição de acentuação no Excel.

4. Aba “ORIENTAÇÕES”

Apresenta os conceitos básicos utilizados na ficha de indicadores: perspectiva, objetivo estratégico, indicador de desempenho, descrição operacional, fórmula, unidade de medida, fonte de dados, periodicidade e polaridade.

Recomenda-se a leitura desta aba antes do preenchimento inicial dos indicadores, para assegurar consistência conceitual em toda a organização.

5. Aba “RESULTADOS”

Apresenta um relatório consolidado com todos os indicadores cadastrados, incluindo: perspectiva, objetivo estratégico, nome do indicador, descrição operacional, fórmula, unidade, fonte de dados, periodicidade, polaridade, gestor responsável e status (rascunho ou definitivo).

  • Filtros por Perspectiva e Objetivo Estratégico.
  • Botão “Imprimir relatório” – pode ser usado para gerar PDF, via opção “Salvar como PDF”.
  • Botão “Exportar para Excel (CSV)” – exporta os dados dos indicadores em formato compatível com o Excel, preservando acentuação.

6. Boas práticas de uso

  • Centralizar o cadastro estrutural de perspectivas, objetivos, indicadores e metas sob responsabilidade do ADMIN.
  • Orientar os gestores a utilizarem sempre o Salvar rascunho enquanto estiverem testando ou calibrando indicadores e metas.
  • Utilizar o Salvar definitivo apenas após validação dos dados em fóruns de governança (comitês, reuniões de diretoria, etc.).
  • Guardar a senha do ADMIN em local seguro e prever política de troca periódica.
  • Exportar periodicamente os dados para arquivos externos (CSV/PDF) para fins de backup e prestação de contas.
